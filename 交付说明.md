# Zeus H5 设计稿交付说明

## 文件清单

| 文件 | 对应页面 |
|------|----------|
| `index.html` | 登录/注册页 |
| `mySettings.html` | 设置页（个人中心） |
| `my.html` | 我的（个人主页） |

## 全局设计参数

- **主题色** `#F19871`，深色 `#E8865A`
- **背景** 微暖渐变 `#FFF8F5 → #FFFFFF`
- **字体** `PingFang SC, Microsoft YaHei`
- **圆角规范** 卡片 16px，按钮 14px，输入框 12px
- **间距节奏** 字段 18px，滑块上距 30px，按钮上距 34px
- **阴影** 卡片 `box-shadow: 0 2px 12px rgba(0,0,0,0.04)`

## index.html — 登录/注册

### 三种登录模式
| 模式 | 切换方式 | 字段 |
|------|----------|------|
| 密码登录 | 默认 | 用户名 + 密码 + 滑块 |
| 验证码登陆 | 底部"验证码登陆" | 手机号 + 短信验证码 + 滑块 |
| 注册 | 顶部 TAB "注册" | 手机号 + 短信 + 姓名 + 密码 + 确认密码 + 滑块 + 协议勾选 |

### 交互要点
- 用户名/密码校验 → 实时，焦点离开时触发
- 滑块验证 → 拖到最右通过，绿色反馈
- 短信倒计时 → 60s，不可重复点击
- 密码显隐 → 眼睛图标切 type
- Toast → 顶部毛玻璃弹入，2s 自动消失

## mySettings.html — 设置页

### 页面结构
- 顶部头像区（昵称首字 + 姓名 + 部门）
- 账号信息卡片：手机号 / 昵称 / 登录密码
- 退出登录（红色描边按钮）

### 弹窗交互
| 功能 | 弹窗内容 | 校验 |
|------|----------|------|
| 修改昵称 | 输入新昵称 | 不能为空，最少 2 字 |
| 修改手机号 | ① 原手机验证码 ② 新手机号 + 验证码 | 两步独立校验，失败弹窗不关 |
| 修改密码 | 原密码 + 新密码 + 确认密码 | 原密码非空，新密码 6-20 位，两次一致 |
| 退出登录 | 确认提示 | 无 |

> 校验失败时弹窗保持不关闭，可原地修改重新提交

## my.html — 我的（个人主页）

### 用户类型自适应

| 字段 | SaaS 用户 | 自然流量用户 |
|------|-----------|-------------|
| 公司名 | ✅ 顶部大字 26px/800 | ❌ 不显示 |
| 公司 Logo | ✅ 替换头像为图片 | ❌ 显示文字头像 |
| 部门 | ✅ 姓名下方 | ❌ 不显示 |
| 角色 | 多角色胶囊标签 | 单角色标签 |

数据在 JS 顶部 `user` 对象中配置，切换类型只需改一行注释。

### 页面结构
- **公司名** — 顶部加粗大字（有则显示）
- **信息行** — 头像/Logo + 姓名 + 手机号 + 铃铛（消息角标）
- **标签行** — 角色胶囊 + 部门文字
- **统计栏** — 累计佣金 / 邀约客户 / 团队成员
- **菜单卡片** — 常用功能（佣金/邀约/团队/邀请好友）+ 其他（关于/设置/联系/消息）
- **底部导航** — 首页/客户管理/拓客空间/产品管理/我的

### 交互要点
- 铃铛角标实时显示未读消息数
- 菜单项点击 Toast 反馈
- 底部导航 Tab 切换高亮

## 集成注意

- 三个 HTML 均为纯静态，CSS/JS 内联，无外部依赖（除 logo 图片引用远程 URL）
- 可直接浏览器打开预览效果和交互
- 集成到 Vue 项目时，参考 HTML 中的 CSS 变量和交互逻辑
- 表单校验、Toast、Dialog 等交互逻辑按现有 Vue 项目规范重写，本文件仅供参考
